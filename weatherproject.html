Site Purpose Statement

The purpose of this site is to deliver the weather in a manner that is easy to read. This site will contain the waeather of 3 different cities so it will need to be easy to navigate, finding city, date, and weather conditions.

Audience Description

The audience for this site will be for almost all different ages over the age of about 12. Young and old are interested in the weather and for this reason it should be extremely easy to read and navigate for those that are not so web savvy. The audience will also be from the 3 different cities that we cover so we cannot favor one city over the other.
